# 🛠 Windows Workflow Optimizer (AutoHotkey v2)

Набор скриптов для превращения стандартной периферии в профессиональный инструмент управления. Оптимизирован для фронтенд-разработки, навигации в стиле macOS и глубокой кастомизации вертикальных мышей (Ugreen M571P и аналоги).

## Основные функции

### 1. 🖱 Прокачка мыши (Mouse Layer)
Использует боковые кнопки для управления рабочим пространством и инструментами разработчика. Логика инвертирована для соответствия естественному движению пальца на вертикальной мыши (57°).
* **Глобальная навигация**: Одиночное нажатие боковых кнопок переключает виртуальные рабочие столы:
    * **Ближняя (XButton1)** — предыдущий стол (`Win + Ctrl + ←`).
    * **Дальняя (XButton2)** — следующий стол (`Win + Ctrl + →`).
* **Безопасность**: `Win + Alt + Любая боковая кнопка` — мгновенная блокировка Windows через `DllCall`.
* **Контекстные действия (Ctrl + Кнопка)**:
    * **Браузеры (Chrome, Vivaldi, Edge, Firefox)**:
        * Ближняя → Переключение мобильного вида (Mobile View).
        * Дальняя → Переключение инструментов разработчика (F12 Toggle).
    * **IDE (VS Code, Zed)**:
        * Ближняя → Вызов AI-панели чата (`Ctrl + Alt + B`).
        * Дальняя → Переключение терминала (`Ctrl + J`).

### 2. ⌨️ Улучшенный Caps Lock (Navigation Layer)
Клавиша Caps Lock превращается в модификатор слоя навигации в стиле macOS/Vim:
* **Tap (< 200 мс)**: Обычное переключение Caps Lock.
* **Hold**: Активирует слой навигации:
    * `Caps Lock + Left / Right` → **Home / End** (в начало/конец строки).
    * `Caps Lock + Up / Down` → **Ctrl + Home / End** (в начало/конец документа).
    * `Caps Lock + Backspace` → Удалить всё до начала строки.
* **Выделение**: Все команды поддерживают `Shift` для быстрого выделения текста.

### 3. 🌐 Мгновенная раскладка (Zero-Latency Switch)
Принудительная фиксация языка вместо циклического переключения:
* **Left Shift (Tap)** → Всегда включает **Русский**.
* **Right Shift (Tap)** → Всегда включает **English**.
* *Прозрачность*: Не мешает работе стандартных сочетаний (`Shift + стрелки` для выделения и др.).

### 4. 🚫 Отключение Insert
Клавиша `Insert` полностью подавляется (`Insert::return`), что предотвращает случайное переключение в режим замены при редактировании текста.

## Технические особенности
* **Группировка процессов**: Редакторы и браузеры вынесены в `GroupAdd`, что позволяет добавлять новые приложения (например, Arc или Sublime) одной строкой.
* **v2-Native**: Код написан на AutoHotkey v2 с использованием хуков (`InstallKeybdHook`), маскировочной клавиши (`A_MenuMaskKey := "vkE8"`) и прямых системных вызовов.
* **Эргономика**: Пространственная логика кнопок мыши настроена на когнитивное соответствие направлению движения на экране.

## Структура проекта
* `index.ahk` — Точка входа, объединяющая все модули.
* `vertical_mouse.ahk` — Логика боковых кнопок мыши и контекстных действий.
* `macos_logic.ahk` — Слой навигации через Caps Lock.
* `language_switch.ahk` — Умное переключение раскладки через Shift.
* `unplug_insert.ahk` — Полное подавление клавиши Insert.

## Установка
1. Установите [AutoHotkey v2](https://www.autohotkey.com/).
2. Скачайте проект и запустите `index.ahk`.
3. Для автозагрузки создайте ярлык файла `index.ahk` в папке `shell:startup`.
