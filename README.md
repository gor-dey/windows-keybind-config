# AutoHotkey V2 Config: Улучшенные клавиши и навигация

Этот проект содержит набор скриптов для AutoHotkey v2, которые оптимизируют работу с клавиатурой в Windows, добавляя удобное переключение языков и навигацию через Caps Lock.

## Основные функции

### 1. Умное переключение раскладки (`language_switch.ahk`)
Вместо стандартных комбинаций используется одиночное нажатие клавиш:
*   **Левый Ctrl** (короткое нажатие) — всегда включает **Русский** язык.
*   **Правый Ctrl** (короткое нажатие) — всегда включает **Английский** язык.
*   *Примечание:* Клавиши продолжают работать как модификаторы (Ctrl+C, Ctrl+V и т.д.) при обычном использовании. Переключение происходит только при одиночном клике.

### 2. Улучшенный Caps Lock (`macos_logic.ahk`)
Клавиша Caps Lock превращается в мощный инструмент навигации:
*   **Короткое нажатие** (< 200 мс): Работает как обычный Caps Lock (включение/выключение заглавных букв).
*   **Удержание Caps Lock**: Активирует слой навигации:
    *   `Caps Lock + Left` → **Home** (в начало строки)
    *   `Caps Lock + Right` → **End** (в конец строки)
    *   `Caps Lock + Up` → **Ctrl + Home** (в начало документа)
    *   `Caps Lock + Down` → **Ctrl + End** (в конец документа)
    *   `Caps Lock + Backspace` → Удалить всё до начала строки.
*   **Выделение текста**: Все навигационные клавиши поддерживают Shift для выделения (например, `Caps Lock + Shift + Left` выделит текст до начала строки).

## Установка

1.  Установите [AutoHotkey v2](https://www.autohotkey.com/).
2.  Скачайте или клонируйте этот репозиторий.
3.  Запустите файл [index.ahk](index.ahk) (основной файл, который объединяет остальные).

## Структура проекта

*   [index.ahk](index.ahk) — точка входа, подключает модули.
*   [language_switch.ahk](language_switch.ahk) — логика переключения языков.
*   [macos_logic.ahk](macos_logic.ahk) — навигация и поведение Caps Lock.

## Автозагрузка
Чтобы скрипт запускался автоматически при старте Windows:
1. Нажмите `Win + R`.
2. Введите `shell:startup` и нажмите Enter.
3. Создайте ярлык для файла `index.ahk` в открывшейся папке.
